## Search Strategy

### PubMed Search Tips

1. **Use Medical Subject Headings (MeSH)** for consistency
2. **Set date range** "Last 4 years" for most recent evidence
3. **Use Boolean operators**: AND, OR, NOT
4. **Save URLs** directly into Q file (use "Share" → "Link")
5. **If paywalled**: Check institutional access OR email authors for reprint

### Search Priority Order

**CRITICAL (do first):**
1. Q6: CSPG4 on immune cells (decision blocker)

**FOUNDATION (next):**
2. Q1: SLE epidemiology baseline
3. Q2: CAR-T proof of concept
4. Q3: Senescence in autoimmunity

**TARGETS (parallel):**
5. Q4, Q5, Q7: Individual target validation

**METHODS (last):**
6. Q8, Q9, Q10: Methodological justification

## Paper Organization

Once found, classify each paper as:

- **CRITICAL** (★★★): Essential for main claim, must cite in manuscript
- **IMPORTANT** (★★): Supports key findings, should cite
- **BACKGROUND** (★): Context only, cite if space permits
